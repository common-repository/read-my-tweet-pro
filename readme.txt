== Description ==

The features of **Read My Tweet Pro** are few, but just what is needed to present your tweets to your visitors in a nice way.

Here are the features of **Read My Tweet Pro**:

1. **3 colors to select**
2. **Adjust the speed** of the scroller
3. **Adjust the width** of the scroller
4. **Select how many tweets** to display
5. **Pause** when hovered over
6. Can be shown in a page template or blog post
7. **Easy to style** with CSS

Checkout the Premium plugin **Read My Tweet Plus** and display tweets, posts or notices/advertisements. Go to plugins page -->  - http://readmytweetpro.wpdolly.com/wordpress-twitter-plugin/read-tweet-plus/


== Installation ==

1. Open your wordpress admin and go to the plugins menu
2. Press add new and then go to the upload menu 
3. Choose the readmytweet.zip file and install it
4. Activate the plugin and go to the plugin settings page
